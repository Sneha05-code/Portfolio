
# 1.Project Overview

# Project Title
Responsive Personal Portfolio Website

# Description
This project is a fully responsive personal portfolio website designed to showcase an individual’s skills, projects, and contact information. It follows modern web development practices, including semantic HTML5, external CSS styling, responsive layouts using Flexbox/Grid, and accessible markup.

# Goals & Objectives
The primary objectives of this project are:
✔ Build a professional personal website ✔ Ensure full responsiveness across devices ✔ Implement clean and semantic HTML structure ✔ Use CSS Flexbox/Grid for layout ✔ Provide a working navigation menu ✔ Include a functional contact form with validation ✔ Follow accessibility best practices
This website serves as a real-world example of a front-end web development project.

# 2.Setup Instructions
Prerequisites
You need:
• A web browser (Chrome, Edge, Firefox, Safari)
• VS Code
• Live Server extension

# Step-by-Step Setup Guide
Step 1 — Create Project Folder
Create a folder named:
Portfolio
 Step 2 — Create Files
Inside the folder, create:
index.html
styles.css
script.js
images/profile.jpg,project1.png

Step 3 — Add HTML Structure
HTML into index.html.

Step 4 — Add CSS Styles
CSS into styles.css.

Step 5 — Add JavaScript
toggle + validation script into script.js.

Step 6 — Run the Website
Either:
✔ open index.html in your browser or ✔ use Live Server in VS Code
Your site will now load.

# 3.Code Structure
# File Hierarchy
portfolio/
│── index.html → main webpage
│── styles.css → styling & layout
│── script.js → navigation + validation logic
│── images/ → profile + project screenshots

# Code Organization
HTML
Uses semantic tags:
•<header> — navigation
•<main> — website content
•<section> — grouped content
•<footer> — closing info

CSS
Includes:
•CSS variables
•Flexbox/Grid layouts
•Media queries
•Transitions
•Form styles and focus states

JavaScript
Handles:
•Mobile hamburger menu toggle
•Form validation
•Footer date auto-update

# 4.Technical Details
Architecture
This is a Front-End Only Web Application built with:
•HTML5 — structure
•CSS3 (Flexbox + Grid) — layout and styling
•JavaScript — interactivity no backend is required.

# Responsive Design Strategy
This project uses mobile-first design:
•Flexbox/Grid auto-adjust layouts
•Images resize proportionally
•Navigation collapses to hamburger menu on small screens
•Media queries adapt tablet & desktop layouts
